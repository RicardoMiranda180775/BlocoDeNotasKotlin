# Bloco de Notas

Bloco de Notas desenvolvido em Kotlin.

## Design Patterns 
App foi implementado com MVVM e Clean architecture usando princios de SOLID.

 - Kotlin
 - Android Jetpack

- Lifecycles

- Navigaton

- Room
- ViewModel
- AppCompat
- Retrofit
- Dagger 2
- Coroutines
